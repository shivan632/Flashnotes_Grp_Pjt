## 📌 Project Overview

FlashNotes is a modern full-stack web application designed to help users learn efficiently through structured notes, flashcards, and quiz-based interactions.

The platform focuses on improving retention and engagement by combining:

- Note creation and management
- Flashcard-based revision
- Interactive quizzes
- Clean and responsive user interface

It provides a seamless experience for both registered users and guest users with offline capabilities.

---

## 🎯 Business Objective

The goal of this project is to build a scalable and user-friendly learning platform that:

- Enhances learning efficiency
- Improves knowledge retention
- Encourages active revision through quizzes
- Provides accessibility across devices
- Supports offline usage for better user experience

---

## 🏗️ System Architecture

The application follows a full-stack architecture:

- Frontend: Handles UI, user interactions, and client-side logic
- Backend: Manages APIs, authentication, and database operations
- Database: Stores user data, notes, and quiz information

The system is designed to be modular, scalable, and maintainable.

---

## 🛠️ Tech Stack

### Frontend
- React (with TypeScript)
- Vite
- Chakra UI
- State management and API integration tools

### Backend
- FastAPI
- SQLModel (ORM)
- Pydantic (data validation)
- REST API architecture

### Database
- PostgreSQL

### Additional Technologies
- JWT Authentication
- IndexedDB (for offline mode)
- Docker (for containerization)
- GitHub Actions (for CI/CD)

## 📊 Core Functional Flow

1. User logs in or continues as a guest  
2. Creates or accesses notes  
3. Converts notes into flashcards  
4. Practices using flashcards  
5. Takes quizzes for evaluation  
6. Tracks learning progress
